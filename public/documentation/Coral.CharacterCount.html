Coral.CharacterCount

Current Version: 3.0.1
Related Components: Coral.Textfield, Coral.Textarea
Github: coralui-component-charactercount

A simple component that indicates to users the remaining characters in a textfield or textarea.

Coral.CharacterCount API

Constructor

JavaScript:

new Coral.CharacterCount() or document.createElement('coral-charactercount')

HTML Tag:

<coral-charactercount>

Extends

  • Coral.Component

Static Properties

Enumeration representing the possible targets for the CharacterCount.

Properties:

Name Type Value Description

NEXT String _next Relates the CharacterCount to the next sibling.
