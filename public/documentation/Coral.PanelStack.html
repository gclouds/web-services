Coral.PanelStack

Current Version: 1.0.3
Github: coralui-component-panelstack

This component holds a collection of panels and shows the selected panel. This component uses the Coral.mixin.selectionList specification.

instance.selected {Boolean}
    

    

    

    




    
    

        Whether the item is selected. When true, the item will appear as the active element in the PanelStack. The item
        must be a child of a PanelStack before this property is set to true. This property cannot be programmatically set
        to false.

  • false
