Coral.Tag

Current Version: 3.0.5
Related Components: Coral.TagList
Github: coralui-component-taglist

Tag is a component used inside a Coral.TagList.

Tag attributes

|

multiline

By default, tags will truncate content if its too big for the tag's container. Add the multiline attribute to support mutli-line text instead of truncating.

instance.remove

    

    






    Non-destructively remove this element. It can be re-added by simply appending it to the document again.
    It will be garbage collected if there are no more references to it.
    This is different from using jQuery's remove(), which is destructive in that it removes data and listeners.
