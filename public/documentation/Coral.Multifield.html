Coral.Multifield

Current Version: 2.0.0
Github: coralui-component-multifield

A Multifield is a composite field that allows you to add, reorder, and remove multiple instances of a component.

Markup Examples

Pre-filled items

A Multifield item can have a pre-filled content different from the Multifield template. But added items will always be generated based on the template.

Child elements of <coral-multifield> can be given a special attribute to enable functionality.

  • [coral-multifield-add] - Click to add an item

instance.items {Coral.Collection}
    

    

    
    Readonly
    

    




    
    

        The Collection Interface that allows interacting with the Coral.Multifield items that the component contains.
        See Coral.Collection for more details.
    

    

    


    

	

	

	

	

	

	

	

  

  

	

  

	

	




    



        
            



    
instance.template {HTMLElement}
    

    

    

    




    
    

        The Multifield template element. It will be used to render a new item once the an element with the attribute
        coral-multifield-add is clicked. It supports script, template and
        textarea tags. While specifying the the template from markup, it should include the
        coral-multifield-template attribute.
