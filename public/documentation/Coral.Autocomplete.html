Remote Suggestions

                  

                    Listen to the coral-autocomplete:showsuggestions event and call event.preventDefault()
                    to stop the default behavior of searching items for suggestions. Then, use event.detail.value to
                    request suggestions from the server. Once you have suggestions, pass an array of suggestion objects to
                    autocomplete.addSuggestions() to show suggestions. Listen to the
                    coral-autocomplete:hidesuggestions event and abort any requests you've made to avoid showing
                    suggestions when the user hidden them.
