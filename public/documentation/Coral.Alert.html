Coral.Alert

Current Version: 3.0.5
Github: coralui-component-alert

Alerts can be used as static indicators of an operation's result, or as messages to highlight information to the user.

Alert Examples

|

Close Button

The Coral.Alert component does not include a close button by default, but you can add it manually. For this scenario, please use a large alert and add the close button to the lower right.

INFO This is is an alert. With information about the currrent issue and the option to close the alert.
